# Ecritum v0.1.0 Community Release

This is the first public Community release of Ecritum.

## Artifact

- Lane: Core
- Runtime support: SCI/Clojure scripting
- Platform slice: macOS arm64
- SwiftPM binary target asset: `EcritumRuntime.xcframework.zip`

## Trust Tier

This Community artifact is not Developer ID signed, notarized, or stapled. It
is intended for developers who are comfortable with the extra macOS friction
that can come with unsigned open-source binary artifacts.

The optional Trusted macOS release tier remains future work and will require
Developer ID signing, notarization, and stapling evidence.

## Verification

The release is accepted only when `release-check-community` passes against the
tag-specific GitHub Releases asset URL and matching SwiftPM checksum.
